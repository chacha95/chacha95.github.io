# Detectron2 리뷰

[Detectron2](https://github.com/facebookresearch/detectron2)란 FAIR(Facebook Artificial Intelligence Research)에서 만든 pytorch 기반 object detection 플랫폼입니다. FAIR에서 학습한 pretrained 모델을 API 형태로 제공합니다. 

> Detectron2

![](https://miro.medium.com/max/4000/0*VbMjGBHMC6GnDKUp.png)



### 왜 Detectron2가 좋은가?

현재 딥러닝 프레임워크는 크게 **tensorflow&keras**와 **pytorch** 진영으로 나뉘어져 있습니다.
